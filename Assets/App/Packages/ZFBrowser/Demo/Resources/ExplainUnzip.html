Extract DemoBrowserAssets.zip to your project folder.

One thing before we start:

For this demo to work, we need some files in MyProject/BrowserAssets/, but Asset Packages can't put things outside MyProject/Assets/. These files are game-local HTML/CSS/JavaScript resources.

Find ZFBrowser/Demo/DemoBrowserAssets.zip and extract it to your project folder so it looks something like this:

MyProject/
	Assets/
		...
	BrowserAssets/
		demo/
			DoorControl.html
			HUD.html
			....
	Library/
	ProjectSettings/
	...

Once that's done, start the demo again and enjoy!
